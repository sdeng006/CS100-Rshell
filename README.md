## Summary

  This program is a command shell which perform the same task as a bash shell. This could perform mutiple commands using "||", "&&" or ";", and exit by typing "exit".
  
  To run the program:
      - go to terminal
      
      - git clone 
      
      - cd rshell
      
      - make
      
      - bin/rshell
